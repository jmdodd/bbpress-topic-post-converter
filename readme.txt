== Description ==



Filters are available for modification of plugin behavior. 

== Installation ==

1. Upload the directory `bbpress-topic-to-post-converter` and its contents to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
